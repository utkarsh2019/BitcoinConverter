# BitcoinConverter

An android app that gets you the bitcoin value for currencies like -

 - US Dollar (USD)
 - Euro (EUR)
 - British Pound (GBP)
 - Chinese Yuan (CNY)
 - Indian Rupee (INR)
 
 
Bitcoin Average API used to calculate the conversion value.
